### 구현된 기능

#### 1. 내 가계부 카드
- 초록색 테두리로 시각적 구분
- 현재 사용 중인 가계부 배지 표시
- 멤버 수 정보 표시 (X/2명)
- 초대 상태별 표시:
  - 초대 없음: 초대 버튼 활성화
  - 수락 대기중: "수락 대기중" + 이메일
  - 수락됨: "수락됨" + 이메일
  - 거부됨: "수락 거부됨" + 초대 버튼 활성화
  - 멤버 풀: "멤버 가득 참"

#### 2. 초대받은 가계부 카드
- 주황색 테두리로 시각적 구분
- pending 상태: 수락/거부 버튼
- accepted 상태: "초대 수락함" 배지

#### 3. 정렬 순서
1. 현재 사용 중인 가계부 (맨 위)
2. 내가 만든 가계부 (생성일 역순)
3. 초대받은 가계부

#### 4. 추가 기능
- 당겨서 새로고침 (Pull to Refresh)
- 빈 상태 UI (가계부 없음)
- 에러 상태 UI (재시도 버튼)
- 거부 확인 다이얼로그
