# 수정 요구사항

## 기능: 목표 달성률 업데이트 문제 해결

### 사용자 요청 요약
총자산이 변경되었는데도 목표 달성률이 변함이 없다. 목표 달성률이 총자산 변경을 제대로 반영하도록 수정.

### UI/UX 요구사항
- [ ] 목표 달성률이 실시간으로 업데이트되어야 함
- [ ] 자산 거래 추가/수정 시 즉시 반영

### 기능 요구사항
- [x] assetGoalCurrentAmountProvider가 총자산 변경 시 자동 업데이트
- [x] getCurrentAmount의 assetType 필터 복구
- [x] Provider 캐싱 문제 해결

### 엣지 케이스 처리
- [x] 목표가 없는 경우 (기존 동작 유지)
- [x] assetType이 null인 경우 (전체 자산 목표)
- [x] categoryIds가 null인 경우 (모든 카테고리 목표)
- [x] 목표 달성률 100% 초과 케이스

### 제약사항

- 데이터베이스 스키마 변경 없음
