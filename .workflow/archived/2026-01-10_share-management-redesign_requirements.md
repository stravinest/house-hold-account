# 수정 요구사항

## 기능: 가계부 및 공유 관리

### 사용자 요청 요약
기존 "공유 관리" 메뉴를 개선하여 가계부 중심의 통합 관리 화면으로 재설계

---

## UI/UX 요구사항

### 1. 메뉴 변경
- [x] "공유 관리" -> "가계부 및 공유 관리"로 이름 변경

### 2. 목록 구조
- [ ] 가계부 목록 형식으로 표시
- [ ] 정렬 순서:
  1. 현재 사용 중인 가계부 (맨 위)
  2. 내가 만든 가계부
  3. 초대받은 가계부

### 3. 시각적 구분
- [ ] 내 가계부: 초록색(#4CAF50) 테두리
- [ ] 초대받은 가계부: 주황색(#FF9800) 테두리

### 4. 가계부 카드 정보
- [ ] 가계부 이름
- [ ] 멤버 정보 (멤버 수, 멤버 목록)

## 기능 요구사항

### 1. 내가 만든 가계부 (오른쪽 영역)

| 상태 | 표시 내용 | 버튼 |
|------|----------|------|
| 초대 안함 | - | 초대 버튼 (활성화) |
| 초대 보냄 (대기중) | "수락 대기중" + 이메일 | 초대 버튼 (비활성화) |
| 초대 수락됨 | "수락됨" + 이메일 | 초대 버튼 (비활성화) |
| 초대 거부됨 | "수락 거부됨" | 초대 버튼 (활성화) |
| 멤버 풀 (2명) | - | 초대 버튼 (비활성화) |

### 2. 초대받은 가계부 (오른쪽 영역)

| 상태 | 표시 내용 | 버튼 |
|------|----------|------|
| 수락 안함 (pending) | - | 수락/거부 버튼 (활성화) |
| 수락함 (accepted) | "초대 수락함" | - |

### 3. 초대 거부 처리
- [ ] 거부 버튼 클릭 시 확인 다이얼로그 표시
- [ ] 확인 후 목록에서 제거
