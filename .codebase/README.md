# 코드베이스 문서

이 디렉토리는 프로젝트의 구조와 패턴을 문서화합니다.
Claude Code가 코드 검색 시 이 문서들을 참조하여 효율적인 탐색을 수행합니다.

## 문서 목록

| 파일 | 설명 |
|------|------|
| `ARCHITECTURE.md` | 전체 아키텍처, 레이어 구조, 의존성 흐름 |
| `DIRECTORY.md` | 디렉토리 구조 및 각 폴더의 역할 |
| `API.md` | API 엔드포인트 목록 및 라우팅 구조 |
| `DATABASE.md` | 데이터베이스 스키마, 엔티티 관계 |
| `MODULES.md` | 주요 모듈/서비스 설명 |
| `PATTERNS.md` | 코드 패턴, 컨벤션, 네이밍 규칙 |

## 문서 업데이트

코드 변경 시 관련 문서도 함께 업데이트하세요.
`codebase-update` 스킬을 사용하면 자동으로 가이드를 받을 수 있습니다.

## 주의사항

- 문서는 간결하게 유지 (핵심 정보만)
- 자동 생성 가능한 정보는 문서화하지 않음
- 문서 형식과 스타일을 일관되게 유지
