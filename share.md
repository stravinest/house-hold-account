거래추가(지출) 시에 
반복주기 를 없음 말고 일이나 월 년으로 선택시에 고정비 체크 박스도 같이 추가 한다. 
고정비를 체크 박스를 선택하면 카테고리가 고정비 카테고리를 적용한다. 
체크 박스에 체크를 하면 해당 부분은 고정비로 들어간다. 고정비는 종료월을 선택해도 되고 선택 하지 않아도 된다. 
아래 하단의 더보기를 선택시에 결제수단 관리 밑에 고정비 관리 항목을 추가한다. 고정비 항목 을 선택하면 두가지 체크 박스가 나온다.
1. 고정비를 지출에 편입 []  -> 해당 부분을 선택 하면 지출 카테고리 항목에 고정비로 잡히게 되서 달력이나 통계에 지출로 잡히게 된다. 선택하지 않으면 지출로 안잡혀서 달력에도 표시가 되지 않는다. 다만 통계 쪽에 카테고리 > 고정비 탭에서 표시가 카테고리 처럼 파이 그래프로와 아래 목록 리스트 형태로 볼수 있게 된다. 또한 추이 > 수입 지출 저축 고정비 추가가 되거 고정비 탭에서 확인가능하다  그럴려면 현재 통계 에서 수입 지출 저축 옆에 고정비 탭을 추가해야한다.
2. 고정비 카테고리  추가 삭제 수정 가능 ( 카테고리나, 결제수단 참고 )
